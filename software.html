Software

A major part of our activities is devoted to software development for phenotype informatics, specifically for ontology curation, ontology annotation of phenotype descriptions, and for integrating computable phenotype descriptions from different sources with other biological data. All our software is open-source, and our code development is hosted on public source code repositories on Github and SourceForge. We welcome contributors, whether to testing, API development, or code. We also actively contribute to several existing open-source projects, including OBD and Phenote.
